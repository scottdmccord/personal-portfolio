Puzzle-d

Puzzle-d is the first application I created during my time in General Assembly's Web Development Immersive. After the user enters in basic information about themselves, he or she is directed to the puzzle board page. Here, I use JavaScript functionality to randomly render an image and then slice it up into individual puzzle pieces. The user clicks to swap pieces. When the puzzle is put back together, the user has won the game!

Neighborhood Eats

Neighborhood Eats is a full stack application with Javascript, Express, and MongoDB. It uses the Yelp API to search for restaurants in 19 New York City neighborhoods. Each search returns the neighborhood's average score for that cuisine type (i.e. the average yelp rating for that search). Ever wonder which neighborhoods had the highest rated cuisines? Now you can find out!

About Me
